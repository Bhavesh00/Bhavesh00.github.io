About the application

The code for the application can be found here . “The Watchman” is a video surveillance tool which helps a given user to perform human face recognition using an existing knowledge base and validate whether the human in the given video is a threat to the user or not. Methodology The way our application achieves the above functionality is making use of image detection features of OpenCV. We detect the various images in the given video stream and try to map them to existing images in the knowledge base.
